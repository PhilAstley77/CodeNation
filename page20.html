1.7 Describe which are the appropriate filetypes for websites

There are many file types you may find on wesites, these include but certainly are not limited to .avi for video, css for the background styling, obviously html for all the content, jpeg or png for photos, mp4 for video, rar and zip for compressed files. There are many more but they all do the same or similar, for instance there are probably hundreds of video file formats, way too many to list!
